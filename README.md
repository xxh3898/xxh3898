## 👋 소개

> **"응답 속도를 96% 단축하는 백엔드 개발자"**  
> 단순 기능 구현을 넘어 **클라우드 인프라와 애플리케이션의 연관성**을 분석해 시스템의 근본적인 병목을 해결합니다. N+1 쿼리로 인한 EC2 CPU 크레딧 고갈 장애를 분석·해결하며, 앱 코드의 비효율이 인프라 자원 마비로 직결될 수 있다는 **거시적인 병목 시야**를 갖춘 백엔드 개발자입니다.

- **성능 병목 분석 및 최적화**: JPA `@Query JOIN FETCH`를 활용해 150개의 단건 쿼리를 3개의 벌크 쿼리로 최적화, EC2 CPU 크레딧 고갈 문제를 동시 해결하며 API 응답 속도를 25초에서 0.9초로 단축했습니다.
- **문서화 중심 개발 습관**: 기획서부터 트러블슈팅까지 개발 템플릿을 직접 제작하여 전 과정을 투명하게 기록하고 지식 자산으로 만듭니다.
- **구조화된 개발 프로세스**: `기획서 → API 명세 → 아키텍처` 순의 표준화된 개발 사이클을 준수하며 사후 추적을 용이하게 합니다.
- **협업 가시성 확보**: GitHub Projects 칸반보드 및 이슈/PR 템플릿 도입으로 팀원 간 작업 현황을 시각화하고 코드 리뷰 효율을 높입니다.
